# Project Rules & Guidelines

1. **Design Integrity**: Semua UI harus menggunakan padding `py-6 px-8`, radius `32px`, dan dark theme.
2. **Naming**: Gunakan `appSettings` dari context untuk semua teks branding.
3. **Database**: Jangan hapus tabel tanpa audit RLS.
4. **Git**: Commit message harus deskriptif (feat/fix/chore).
